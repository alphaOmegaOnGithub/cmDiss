A most recent template is available on github: https://github.com/latextemplates/uni-stuttgart-dissertation-template
It uses pdflatex and has many useful packages.

-> Each time something is changed in the bibliography and for initial document creating execute the "createDVI" script
-> To create the glossaries, i.e., the abbreviations list execute the "makeglossaries" script and afterthat run latex. 
Be aware, that "makeglossaries" should be executed after the initial "createDVI" script!
